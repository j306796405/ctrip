服务优势

未消费随时退
退款流程
团购成功但未消费此券 → 登录“我的携程” → 在“团购订单”中“申请退款” → 审核通过后退款至支付账户
退款时间
客人提出退款申请后，携程工作人员将在3个工作日内进行审核；审核通过后，携程将为您进行退款。退款申请将在2个工作日内提交至银行，各银行会根据其业务流程，在2-5个工作日内退款至您的支付账户。请您注意查收（此服务不收取任何手续费）。
以下情况不适用
1. 已消费的携程团购券（包括：在线预约成功的，电话预约成功的，已经使用或入住的）
2. 在产品说明中明确标明不支持“未消费，随时退”的产品或服务
券过期未消费，无条件返还
退款流程
团购券过期且未消费此券 → 登录“我的携程” → 在“团购订单”中“申请退款” → 审核通过后退款至支付账户
退款时间
客人提出退款申请后，携程工作人员将48小时内进行审核；审核通过后，携程将为您进行退款。退款申请将在2个工作日内提交至银行，各银行会根据其业务流程，在2-5个工作日内退款至您的支付账户。请您注意查收（此服务不收取任何手续费）。
以下情况不适用
1. 已消费的携程团购券（包括：在线预约成功的，电话预约成功的，已经使用或入住的）
2. 在产品说明中明确标明不支持“券过期未消费，无条件退款”的产品或服务
消费不满意，携程来免单
退款流程
商家未兑现服务承诺 → 立刻联系携程客服10106666 → 经携程客服核实确认后 → 给予立即退款
保障适用范围
1. 商家提供给携程会员的产品或服务与其承诺的产品或服务内容严重不符
2. 商家提供的产品出现严重质量问题，或有违反《中华人民共和国消费者权益保护法》，《中华人民共和国产品质量法》的情况
